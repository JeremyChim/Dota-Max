[教程] 如何手动安装 / 开启增强模式机器人
[耐心一点，请花5分钟认真阅读如何安装]

[重新打字太累，内容由ChatGPT翻译]

以下内容是教程，帮助你：1. 启用增强模式；2. 解决机器人变成随机名称的默认机器人的问题。

前提条件
该脚本目前仅在 "自定义大厅" 中有效，并将服务器位置设置为 "本地主机"。

如果脚本安装正确，游戏中的机器人名称应该带有 ".OHA" 后缀，例如：OG.ana.OHA

更强的机器人 - 启用 Fretbots 模式
如果你觉得现有的机器人脚本不够刺激，你可以通过以下步骤启用 Fretbots 模式，为机器人提供巨大优势，使游戏更具挑战性。

1. 首先，将此脚本放入正确的文件夹
1. 在Dota2 客户端里面订阅 Open Hyper AI ，确保脚本已经下载完成。
2. 打开文件夹（ Steam\steamapps\workshop\content\570\3246316298\Install-to-vscript ），双击运行文件："quick-install-oha-cn.bat"
3. 如果你不知道这个Steam文件夹的位置，请在Steam库中右键点击Dota2，选择属性 > 已安装文件 > 浏览。" 这时会打开文件夹: "Steam\steamapps\common\dota 2 beta"，将地址中的 "common\dota 2 beta" 部分替换成 "workshop\content\570\3246316298\Install-to-vscript"，按下回车键即可打开正确的文件夹。
4. 如果有Window 安全保护提醒，点击 "更多" > “运行”

通过这样做，任何脚本内容的更新都会自动提供到你的本地脚本文件夹里，无需再次复制。当然，你也可以选择自己创建脚本快捷链接，具体查看下方附录。

2. 最后，在游戏中启用 Fretbots 模式
1. 创建大厅游戏时，确保勾选 启用作弊。将服务器位置选择为 本地服务器。
2. 游戏开始时（例如在英雄选择阶段），打开控制台，输入以下命令并按回车：
sv_cheats 1; script_reload_code bots/fretbots
3. 如果游戏中的机器人变回默认机器人，你应选择 本地开发脚本 作为机器人的脚本。
4. 启用成功时，游戏中会有提示音和文本通知。示例：https://steamcommunity.com/sharedfiles/filedetails/?id=3289405415
5. 如果你不知道如何在 Dota2 中使用控制台：在 Steam 中编辑 Dota2 属性，在 常规 > 启动选项 中添加 -console，然后关闭窗口并启动 Dota2。或者观看此视频了解如何使用控制台：https://www.youtube.com/watch?v=d7k2plNvzBM


以上操作你只需要做一次。以后每局新游戏，你都只需要打开控制台，输入那一行代码即可。

附录 - 其他安装方式：

方案1 - 自动安装并保持脚本更新，只需运行一个命令，非常简单。
  1. 打开预装的 Windows 应用程序：CMD 或 PowerShell。通过按键盘上的“Windows”键并输入“cmd”来打开它，右键单击以管理员身份运行。
  2. 使用 `mklink` 命令创建一个快捷方式/符号链接，将引号中的文本替换为您的实际文件夹路径：
    mklink /d "<Steam_local_bot_dev_folder>" "<Steam_workshop_item_folder>"
  3. 例如，我在 C:\ 安装了 Dota2：
    mklink /d "C:\Program Files (x86)\Steam\steamapps\common\dota 2 beta\game\dota\scripts\vscripts\bots" "C:\Program Files (x86)\Steam\steamapps\workshop\content\570\3246316298"

通过这样做，您无需手动复制/粘贴内容。工作坊内容文件夹中的任何更新都会同步到本地开发机器人文件夹，因为它们指向同一位置。

方案2 - 通过手动复制文件来安装一个固定的版本，详见此指南：https://steamcommunity.com/workshop/filedetails/discussion/3246316298/4520010047766285327/

附录 - 故障排除：

  1. 如果你不知道 Steam 文件夹的位置，右键点击 Steam 库中的 Dota2，选择属性 > 已安装的文件 > 浏览。
  2. 执行`mklink` CMD命令需要管理员权限，因此你需要右键点击 CMD 或 PowerShell 应用程序并选择 以管理员身份运行 来执行此命令。
  3. 如果由于冲突导致命令失败，首先需要重命名或删除 vscript 目录中已经存在的机器人文件夹，然后再次运行 mklink 命令来创建快捷方式。
  4. 如果文件夹名称中有空格，请确保将整个路径放在引号中。
  5. 确保在 Dota2 客户端内订阅此脚本，并且 Dota2 已经将工作坊内容下载到你的磁盘中，只有这样你才能复制或创建有效的快捷链接。

附录 - 什么是 Fretbots 模式

关于为什么应该尝试 Fretbots 的附加说明。
> 或者 Fretbots 到底具体做了什么。

  1. Fretbots 在机器人死亡时以及每隔一分钟都会为机器人提供额外奖励。
  2. 启用 Fretbots 后，会要求所有玩家为难度投票，投票结果将影响机器人获得的奖励数量。
  3. 可能的奖励包括：金币、经验值、护甲、生命/魔法恢复、属性点数。你投票的难度越高，机器人获得的奖励就越多。如果选择难度超过 5，游戏开始约 5 分钟后，机器人的 GPM 将永远高于玩家。
  4. 请注意，如果玩家团队中的人类玩家技能水平较高，敌方机器人将获得更多的奖励，因为机器人获得的奖励会根据对方团队中相同角色的玩家表现动态调整。
  5. 这个模式的乐趣在于，通常机器人在对线阶段的表现较差，因为开发者很难使用 Valve 当前的机器人 API 实现良好的对线技能。而启用 Fretbots 后，机器人会在中后期游戏中表现得更出色，因为它们的发育会很好。这为玩家提供了额外的游戏体验，早期游戏较为简单，中期游戏有更多挑战，需要玩家更多的团队合作和策略才能完成游戏。
  6. 此外，使用 Fretbots 模式可以解决机器人不懂如何选择或使用中立物品的问题。

附录 - 当前 Fretbots 模式规则

  1. 你可以为难度等级从 0 到 10 进行投票，难度等级影响机器人获得的奖励数量。
  2. 如果难度等级 >= 0，机器人会获得中立物品奖励，并且每分钟获得金币、经验值、属性等公平的奖励。
  3. 如果难度等级 >= 1，机器人在死亡时获得上述奖励；并且还会获得新的魔法/生命恢复奖励。
  4. 随着难度等级的增加，机器人会更早获得中立物品，并获得更高的奖励。
  5. 如果难度等级 >= 5，当玩家杀死机器人时，击杀玩家的金币奖励会减少，但不影响助攻的玩家。机器人死亡时提供的经验值也会减少。
  6. 你投票的难度越高，机器人获得的奖励就越多，从而使游戏更具挑战性。
  7. 难度投票结束后，你可以为盟友机器人的奖励比例相对于敌方机器人进行投票。例如，如果你投票为 0.5，意味着你的盟友机器人获得的奖励仅为敌方机器人的一半——这样你可以使游戏对自己更具挑战性。

附录 - Fretbots 模式难度建议

  1. 高难度可能会令人不知所措甚至令人沮丧，请为你和你的团队选择适合的难度，以便在游戏中获得更多乐趣！
  2. 难度 0 适合新玩家或想轻松享受游戏的玩家，难度 2 适合与朋友一起进行休闲游戏，难度 5 开始变得紧张，要求团队有良好的合作或使用一些强大的英雄，难度 10 是极限，你需要一些特定的强力英雄并且团队有良好的合作或策略才能获胜。
  3. 请注意，Fretbots 模式的主要目的是鼓励团队合作！
  4. Fretbots 的难度是动态的。意思是，机器人会根据相应角色玩家的表现获得额外的奖励。团队中的玩家表现越好，机器人获得的奖励就越多。例如，难度 5 通常会使机器人获得较高的 GPM，如果玩家的平均 GPM 为 600，机器人可能会达到 700 GPM，如果玩家为 800 GPM，机器人可能会达到 1000 GPM。
